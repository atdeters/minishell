*NAME*
    cd - Change the shell working directory

*USAGE*
    cd [relative or absolute path to directory]

*FLAGS*
    --help / -h         display this help

*Examples"
    cd ~                change working directory to home
    cd                  change working directory to home
    cd ..               goes up folder structure by one level
    cd ~/minishell      change working directory to the minishell directory in home
