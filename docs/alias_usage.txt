*NAME*
    alias â€“ Create, list, or remove command aliases, allowing command substitution with support for pipes and redirections.

*USAGE*
    alias [flags] alias[=expansion]

*FLAGS*
    --remove / -r       Removes an alias (if it already exists)

*EXAMPLES*
    alias               Display all available aliases
    alias    [name]     Display the aliases with the corresponding expansion
    alias -r [name]     Remove a specific alias
    alias -h            Display this help

*NOTE*
    It expects a single command as input. So make sure to format it with quotes.
    - 'alias "st=git status"' instead of 'alias st=git status' or 'alias st="git status"'
